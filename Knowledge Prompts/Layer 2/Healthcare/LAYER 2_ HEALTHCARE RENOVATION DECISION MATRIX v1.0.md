## 簡介

### 目的

Healthcare Renovation Decision Matrix 提供了一個結構化的決策工具，幫助 GC 快速確定項目的關鍵特性，特別是在 Stage 1 初始驗證中。

### 使用場景

- **Stage 1**: 初始驗證 - 確定改造強度等級、複雜度評分等
- **Stage 2**: 網站評估 - 確認地區特定因素
- **Stage 3**: 初步估算 - 確定調整因子

### 核心問題

通過回答 10 個關鍵問題，可以快速確定：
1. 改造強度等級 (Level 1-4)
2. 項目複雜度評分 (8-16 分)
3. 主要風險因素
4. 初步成本範圍

## 10 問快速評估工具 (v1.0)

### 問題 1: MEP 系統重用計劃

**問題**: 現有 MEP 系統中有多少百分比可以重用或翻新？

**選項**:
- [ ] **(A) 80-100% 重用** → MEP 重用因子: 0.80-1.00 → 改造等級傾向 Level 1
- [ ] **(B) 50-80% 重用** → MEP 重用因子: 0.50-0.80 → 改造等級傾向 Level 2
- [ ] **(C) 20-50% 重用** → MEP 重用因子: 0.20-0.50 → 改造等級傾向 Level 3
- [ ] **(D) 0-20% 重用** → MEP 重用因子: 0.00-0.20 → 改造等級傾向 Level 4

**說明**: 
- Level 1 (化妝改造): 大部分 MEP 系統保留
- Level 2 (中等改造): 部分 MEP 系統更換
- Level 3 (主要改造): 大部分 MEP 系統更換
- Level 4 (完全重建): 幾乎所有系統更換

---

### 問題 2: 結構系統加固

**問題**: 現有結構系統需要多少程度的加固或升級？

**選項**:
- [ ] **(A) 無加固** → 結構重用因子: 0.95-1.00 → 改造等級傾向 Level 1
- [ ] **(B) 輕微加固** → 結構重用因子: 0.85-0.95 → 改造等級傾向 Level 1-2
- [ ] **(C) 部分加固** → 結構重用因子: 0.70-0.85 → 改造等級傾向 Level 2-3
- [ ] **(D) 主要加固** → 結構重用因子: 0.50-0.70 → 改造等級傾向 Level 3-4
- [ ] **(E) 完全重建** → 結構重用因子: 0.00-0.50 → 改造等級傾向 Level 4

**說明**:
- 無加固: 結構完好，無需升級
- 輕微加固: 地震加固、部分補強
- 部分加固: 次要結構工作、局部加固
- 主要加固: 廣泛結構工作、重要構件升級
- 完全重建: 新結構或 90%+ 更換

---

### 問題 3: 環保風險評估

**問題**: 現有建築的環保狀況如何？

**選項**:
- [ ] **(A) 無環保問題** → 環保調整: 0% → 成本影響: 無
- [ ] **(B) 輕微污染** → 環保調整: +1-2% → 成本影響: $50-150K
- [ ] **(C) 中等污染** → 環保調整: +2-5% → 成本影響: $150-500K
- [ ] **(D) 重大污染** → 環保調整: +5-10% → 成本影響: $500K-$1M+
- [ ] **(E) 未知** → 環保調整: +3-5% → 成本影響: $150-300K (風險預留)

**說明**:
- 輕微污染: 石棉、含鉛油漆、輕微黴菌
- 中等污染: 多個污染源、需要專業整治
- 重大污染: 嚴重污染、需要大規模整治
- 未知: 需要進行環保評估

---

### 問題 4: 施工期間佔用

**問題**: 施工期間設施是否需要繼續運營？

**選項**:
- [ ] **(A) 完全關閉** → 複雜度調整: 0 → 工作方式: 標準施工
- [ ] **(B) 部分分階段佔用** → 複雜度調整: +2 → 工作方式: 分區施工
- [ ] **(C) 完全運營** → 複雜度調整: +3 → 工作方式: 24/7 協調

**說明**:
- 完全關閉: 無患者、無員工，標準施工方式
- 部分分階段: 部分區域運營，需要分區施工
- 完全運營: 設施持續運營，需要最小化干擾

---

### 問題 5: 行為健康特殊設計要求

**問題**: 項目是否需要行為健康特殊設計？

**選項**:
- [ ] **(A) 完整** → 防綁紮、防篡改、患者監督 → 複雜度調整: +2 → 成本溢價: +5-8%
- [ ] **(B) 部分** → 部分要求 (例如防綁紮但無患者監督) → 複雜度調整: +1 → 成本溢價: +2-4%
- [ ] **(C) 標準** → 無特殊要求 → 複雜度調整: 0 → 成本溢價: 0%

**說明**:
- 完整: 所有行為健康設計要求
- 部分: 部分要求
- 標準: 標準醫療設施要求

---

### 問題 6: 地震風險

**問題**: 項目位置的地震風險等級是多少？

**選項**:
- [ ] **(A) 低風險** → 地震設計調整: 0% → 複雜度調整: 0
- [ ] **(B) 中等風險** → 地震設計調整: +2-3% → 複雜度調整: +1
- [ ] **(C) 高風險** → 地震設計調整: +3-5% → 複雜度調整: +2
- [ ] **(D) 非常高風險** → 地震設計調整: +5-8% → 複雜度調整: +3

**說明**:
- 低風險: 地震活動少
- 中等風險: 偶發地震
- 高風險: 頻繁地震、已知斷層附近
- 非常高風險: 高風險地區、特殊設計要求

---

### 問題 7: 地質風險 (液化、地基等)

**問題**: 項目位置的地質風險如何？

**選項**:
- [ ] **(A) 低風險** → 地質調整: 0% → 成本影響: 無
- [ ] **(B) 中等風險** → 地質調整: +2-5% → 成本影響: $100-300K
- [ ] **(C) 高風險** → 地質調整: +5-10% → 成本影響: $300-600K
- [ ] **(D) 非常高風險** → 地質調整: +10-20% → 成本影響: $600K-$1M+

**說明**:
- 低風險: 穩定土壤、無液化風險
- 中等風險: 某些地質風險、可能需要改良
- 高風險: 液化風險、地下水高、需要特殊基礎
- 非常高風險: 嚴重地質問題、需要大規模改良

---

### 問題 8: 建築規範要求

**問題**: 項目適用的主要建築規範是什麼？

**選項**:
- [ ] **(A) 標準建築規範** → 規範溢價: +5-8% → 複雜度調整: 0
- [ ] **(B) OSHPD 3 (醫療設施)** → 規範溢價: +8-12% → 複雜度調整: +1
- [ ] **(C) OSHPD 1 (高風險醫療)** → 規範溢價: +12-18% → 複雜度調整: +2
- [ ] **(D) 多個司法管轄區** → 規範溢價: +10-20% → 複雜度調整: +2

**說明**:
- 標準: CBC、Title 24 等
- OSHPD 3: 一般醫療設施
- OSHPD 1: 高風險醫療設施 (手術室等)
- 多個司法管轄區: 跨州或複雜的監管環境

---

### 問題 9: 設計完成度

**問題**: 設計文件的完成度如何？

**選項**:
- [ ] **(A) 100% 完成** → 設計風險: 0% → 複雜度調整: 0
- [ ] **(B) 90-99% 完成** → 設計風險: +1-2% → 複雜度調整: +1
- [ ] **(C) 70-90% 完成** → 設計風險: +2-5% → 複雜度調整: +2
- [ ] **(D) <70% 完成** → 設計風險: +5-10% → 複雜度調整: +3

**說明**:
- 100% 完成: 完整的施工文件
- 90-99%: 大部分完成，細節待定
- 70-90%: 概念設計完成，詳細設計進行中
- <70%: 早期設計階段

---

### 問題 10: 預期變更單風險

**問題**: 基於項目特性，您預期變更單的風險等級是多少？

**選項**:
- [ ] **(A) 低** → 變更單風險: 0-3% → 應急預留: 5-8%
- [ ] **(B) 中等** → 變更單風險: 3-8% → 應急預留: 8-12%
- [ ] **(C) 高** → 變更單風險: 8-15% → 應急預留: 12-18%
- [ ] **(D) 非常高** → 變更單風險: 15%+ → 應急預留: 18-25%

**說明**:
- 低: 設計完整、業主明確、少變更
- 中等: 標準風險、可能有小變更
- 高: 設計不完整、業主可能變更、複雜項目
- 非常高: 高風險項目、多個不確定因素

## 改造強度等級定義

### Level 1: 化妝改造 (Cosmetic Renovation)

**特徵**:
- MEP 重用: 70-100%
- 結構重用: 90-100%
- 外殼重用: 85-100%
- 主要工作: 內部裝修、表面更新

**成本影響**:
- 基準成本/SF: 0.80x - 1.00x (相對於新建)
- MEP 成本: 30-40% of 新建
- 勞工成本: 40-50% of 新建

**示例**: 
- 重新粉刷、地板更新、照明升級
- 部分 HVAC 更新
- 現有結構保留

**Colton Hospital II**: Level 1 (75% MEP 重用)

---

### Level 2: 中等改造 (Moderate Renovation)

**特徵**:
- MEP 重用: 50-70%
- 結構重用: 70-90%
- 外殼重用: 70-85%
- 主要工作: 50% MEP 更換、部分結構加固

**成本影響**:
- 基準成本/SF: 1.00x - 1.35x (相對於新建)
- MEP 成本: 50-70% of 新建
- 勞工成本: 60-80% of 新建

**示例**:
- 50% HVAC 系統更換
- 給排水系統部分更新
- 結構地震加固

---

### Level 3: 主要改造 (Major Renovation)

**特徵**:
- MEP 重用: 20-50%
- 結構重用: 50-70%
- 外殼重用: 50-70%
- 主要工作: 70% MEP 更換、廣泛結構工作

**成本影響**:
- 基準成本/SF: 1.35x - 1.70x (相對於新建)
- MEP 成本: 70-90% of 新建
- 勞工成本: 80-100% of 新建

**示例**:
- 70% HVAC 系統更換
- 給排水系統大幅升級
- 廣泛結構加固

---

### Level 4: 完全重建 (Complete Reconstruction)

**特徵**:
- MEP 重用: 0-20%
- 結構重用: 0-50%
- 外殼重用: 0-50%
- 主要工作: 90%+ 系統更換、主要重建

**成本影響**:
- 基準成本/SF: 1.70x - 2.00x (相對於新建)
- MEP 成本: 90-100% of 新建
- 勞工成本: 100-120% of 新建

**示例**:
- 完全 MEP 系統更換
- 主要結構重建
- 新外殼

## 決策樹

```
開始
  ↓
問題 1: MEP 重用 %
  ├─→ 80-100% (A) → Level 1 傾向
  ├─→ 50-80% (B) → Level 2 傾向
  ├─→ 20-50% (C) → Level 3 傾向
  └─→ 0-20% (D) → Level 4 傾向
  ↓
問題 2: 結構加固
  ├─→ 無/輕微 (A/B) → 確認 Level 1-2
  ├─→ 部分 (C) → 確認 Level 2-3
  └─→ 主要/完全 (D/E) → 確認 Level 3-4
  ↓
問題 3-10: 其他因素
  ├─→ 環保風險
  ├─→ 施工期間佔用
  ├─→ 特殊設計要求
  ├─→ 地震風險
  ├─→ 地質風險
  ├─→ 建築規範
  ├─→ 設計完成度
  └─→ 變更單風險
  ↓
計算複雜度評分
  ├─→ 基礎分數 (建築類型 + MEP + 結構 + 規範)
  └─→ 調整分數 (風險因素)
  ↓
確定改造等級和複雜度
  ↓
完成
```

## 評分系統

### 複雜度評分計算

**基礎分數** (8 分起點):

```
Building Type Score:
  - Healthcare: 3-4 分
  - Warehouse: 2-3 分
  - Commercial: 2-3 分

MEP Complexity Score:
  - Low (Level 1): 1 分
  - Medium (Level 2): 2 分
  - High (Level 3): 3 分
  - Very High (Level 4): 4 分

Structural Complexity Score:
  - Minimal: 1 分
  - Partial: 2 分
  - Significant: 3 分
  - Major: 4 分

Regulatory Complexity Score:
  - Standard: 1 分
  - OSHPD 3: 2 分
  - OSHPD 1: 3 分
  - Multiple Jurisdictions: 4 分

基礎分數 = 建築類型 + MEP + 結構 + 規範
```

**調整分數** (根據風險因素):

```
Occupancy During Construction:
  - Complete Closure: 0 分
  - Phased Occupancy: +2 分
  - Full Operation: +3 分

Design Completeness:
  - 100% Complete: 0 分
  - 90-99% Complete: +1 分
  - 70-90% Complete: +2 分
  - <70% Complete: +3 分

Environmental Risk:
  - None: 0 分
  - Minor: +1 分
  - Moderate: +2 分
  - Major: +3 分

Seismic Risk:
  - Low: 0 分
  - Medium: +1 分
  - High: +2 分
  - Very High: +3 分

調整分數 = 佔用 + 設計 + 環保 + 地震 (最多 +12 分)
```

**總複雜度評分**:

```
Complexity Score = 基礎分數 + 調整分數
範圍: 8-20 分 (限制在 8-16 分)

評級:
  8-10 分: Low Complexity (1.2x 乘數)
  11-13 分: Medium Complexity (1.5x 乘數)
  14-15 分: High Complexity (1.8x 乘數)
  16+ 分: Very High Complexity (2.0x 乘數)
```

## 翻新成本因子應用

### 醫療保健翻新成本乘數

根據 **RENOVATION_COST_FACTOR_MATRIX_v1.0.yaml**，醫療保健翻新項目的成本乘數如下：

#### 基礎翻新因子: 2.2x

醫療保健翻新成本 = 新建築成本 × 翻新因子 × 地區調整

#### 按翻新範圍調整

| 翻新範圍 | 因子 | 說明 | 典型工作 |
|---------|------|------|--------|
| **Light** | 1.5x | 簡單更新、最少系統更換 | 水管、電氣、照明升級 |
| **Moderate** | 2.2x | 部分系統更換、醫學標準升級 | 部分 MEP 更換、醫學標準升級 |
| **Heavy** | 2.8x | 完整系統更換、醫學標準重新設計 | 完整 MEP 更換、醫學標準重新設計 |

#### 按地區調整

| 地區 | 調整 | 勞工指數 | 說明 |
|------|------|--------|------|
| CA Inland | 0.95x | 1.15 | 內陸加州，勞工成本中等 |
| CA Coastal | 1.10x | 1.25 | 沿海加州，勞工成本較高 |
| CA Bay Area | 1.15x | 1.35 | 灣區，勞工成本最高 |
| TX Coastal | 0.90x | 1.05 | 德州沿海，勞工成本中等 |
| TX Inland | 0.85x | 1.00 | 德州內陸，勞工成本最低 |
| NY Urban | 1.20x | 1.40 | 紐約城市，勞工成本高 |
| FL Statewide | 0.88x | 1.02 | 佛州，勞工成本中等 |

#### 成本驅動因子

醫療保健翻新的主要成本驅動因子：

| 驅動因子 | 成本影響 | 說明 |
